# ALE-Sync-Pro
Utilitaire de synchronisation par lot pour Avid Media Composer

# Qu’est ce que ALE Sync Pro ?
ALE Sync Pro est un utilitaire de synchronisation pour Avid conçu pour accélérer le travail d’assistant monteur.
Bien qu’ALE Sync Pro soit encapsulé dans une application Automator qui le rend plus fonctionnel sur mac, son script sous-jacent en python peut être exécuté en ligne de commande sur PC.
ALE Sync Pro tire parti de la fonctionnalité Avid Log Exchange (ALE), un standard d’échange déjà ancien mais puissant qui permet d’automatiser certaines tâches répétitives de manière externe au logiciel Avid Media Composer.

# À quel usage est destiné ALE Sync Pro ?
ALE Sync Pro permet de synchroniser automatiquement les rushes image aux rushes son sur de grandes quantités de rushes quand ceux-ci disposent d’un timecode commun, lorsque par exemple les cameras sont asservies à l’enregistreur son sur le plateau de tournage.
